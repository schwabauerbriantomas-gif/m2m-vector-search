# Security Policy

## Supported Versions

Only the latest version of the project is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0.x | :white_check_mark: |
| < 1.0.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within M2M Vector Search, please send an e-mail to the maintainers or open a sensitive issue on GitHub. 

We take all security vulnerabilities seriously and will work to resolve them as quickly as possible. Please do not discuss potential vulnerabilities in public forums or issues until the maintainers have had a chance to verify and address the issue.
